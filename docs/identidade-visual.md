# Identidade Visual do Sistema

## Estilo

O sistema segue um estilo:
- minimalista
- moderno
- limpo
- com leve toque tecnológico (neon suave)

O objetivo é manter uma interface bonita, mas sem exageros, garantindo boa leitura e usabilidade para todos os usuários.

---

## Cores

### Fundo
- Fundo principal: #0F172A (preto suave)

### Cores principais
- Azul destaque (neon suave): #22D3EE
- Azul secundário: #0EA5E9

### Texto
- Texto principal: #E2E8F0 (branco suave)

### Estrutura
- Fundo de cards/tabelas: #1E293B

---

## Cores de ação

- Sucesso (confirmar/enviar): #22C55E (verde)
- Erro (cancelar/negar): #EF4444 (vermelho)

---

## Tipografia

- Fonte principal: Inter
- Alternativa: Montserrat

A fonte deve ser simples, moderna e fácil de ler.

---

## Componentes

### Botões
- Bordas arredondadas (8px)
- Cores:
  - Azul (principal)
  - Verde (ações positivas)
  - Vermelho (ações negativas)
- Hover:
  - leve brilho (neon suave)
  - leve mudança de cor

---

### Cards / Caixas
- Fundo: #1E293B
- Sombra leve
- Espaçamento interno (padding) confortável
- Layout organizado e limpo

---

### Tabelas (IMPORTANTE)
- NÃO usar branco puro
- Usar fundo escuro (#1E293B)
- Linhas com leve contraste
- Hover nas linhas com leve destaque azul
- Texto sempre claro para manter legibilidade

---

### Hover (Interações)
- Efeito de brilho leve (glow azul)
- Sem exageros
- Transições suaves

---

## Objetivo Visual

Criar uma interface:
- moderna
- agradável visualmente
- fácil de usar
- organizada
- confortável para uso contínuo

O sistema deve ajudar os líderes a montar escalas com rapidez e clareza, sem poluição visual.
